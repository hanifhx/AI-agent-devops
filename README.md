# DevOps AI Agent
AI-powered DevOps assistant that helps automate infrastructure tasks, debugging, and deployment workflows using FastAPI and OpenAI.

## Overview
This project is an AI-powered DevOps assistant designed to help developers automate DevOps workflows such as infrastructure setup, troubleshooting, log analysis, and deployment guidance.

The agent uses FastAPI as the backend and integrates with OpenAI APIs to generate intelligent responses for DevOps-related tasks.

## Features

- AI powered DevOps assistant
- Infrastructure automation guidance
- Log analysis and debugging help
- Docker & deployment recommendations
- API based interaction using FastAPI
- Modular architecture for easy extension

- ## Tech Stack

- Python
- FastAPI
- OpenAI API
- Docker (planned)
- GitHub
